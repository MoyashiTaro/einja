アソビル　マルチスポーツコート

スポーツ塾ヒーローズアカデミー様のランディングページ制作案件です。 デザイン~コーディングまで担当しました。ランディングページでで最も重要なのは、離脱率を下けげることです。しっかり読まなくても内容が入るようイラストやアニメーション効率的に利用しています。またわかりづらく整理されていなかった一部内容を、一覧にしわかりやすいデザインを心がけ作りました。

UI･UX設計 / WEBデザイン / WEB制作 / SEO対策 / 広告運用 / WEB解析

PLUS株式会社様 HP作成

不動産会社である「PLUS株式会社」様のHP制作案件です。 SUUMOなどのポータルサイトから流入したお客様をターゲットに、企画デザインを行いました。他の不動産会社様との差別化、離脱率低減の2つを目的に制作しております。ホームページの顔であるトップページの写真埋め込みを少し工夫し、インパクトある印象を与えるようデザインしています。ページ構成を非常にシンプルにし無駄な情報を排除、使いやすくすることで離脱率を防ぐ工夫をしております。リリース1ヶ月で、こちらのホームページから１件のご成約をいただきました。

UI･UX設計 / WEBデザイン / WEB制作

K2Mコンサルティング株式会社

新しく設立する会社のホームページ作成の依頼。 デザインからコーディングサーバーアップまで一連の流れを行いました。 企業コンセプトの人を繋げるということを橋のピクチャーを使うことで表し、水色ベースで全体系に爽やで信頼感のあるテイストを意識しました。電話や直接何度もお会いして細かいヒアリングを行ったことでお客様のイメージをデザインに落とし込めるように工夫しました。

UI･UX設計 / WEBデザイン / WEB制作

Shoespicks | スパイク比較

ランニングスパイクの比較検討機能をトリガーとしてEC斡旋を行なっております。 こちらの流入口は検索をメインとして注力し、SEOに力を入れております。「商品名×サブワード」では各商品ごとに検索順位1位を取得しています。Google検索エンジンへの対応と、ユーザーに必要なコンテンツ力でマーケティングに頼らない自然流入の仕組みを構築しました。結果　月間:2万PV CVR:1.16%

BTSドール

BTSのプロモーション動画を作ったよ。 BTSって知ってる？韓国のアイドルグループだよ。

動画制作

モーショングラフィックススクール卒業課題

3DCGを実写に合成した作品です。地球と宇宙はAffter Effectsを利用した3DCGです。 Technology×Artをイメージした作品で、自然の美しさの中の科学を、数式で表現しています。 CG が実写映像を邪魔せぬよう、それぞれの減少に合わせた出現エフェクトをつけることで、CGと映像をなじませています。 音楽はLogic PROを使い自分で制作した曲です。

動画制作

Bynd ”6seconds Challenge

Illustratorを用いて作った素材をAfter Effectsで動きを加えた作品です。Illustratorを用いて作った無機質な素材を、細かい動きの緩急を使って有機的に動かすことに挑戦しています。

動画制作

Bynd ”6seconds Challenge

グラフなどの情報を伝えるツールが動き出す「Infographics」という種類の映像表現です。グラフを動かし、より注意を引きつけかつ分かりやすく伝えることが目的です。この動画では、スマートフォンアプリの使用割合(データは今回の映像のために用意したもので、実際のものではありません）の内訳と、その過半数がYoutubeとNetflixであることを伝えています。

動画制作

Shoespicks

私自身が運営しているWebサービス「Shoes Picks」のPromotionのために作った動画です。ビジネスコンテスト出展を機に作成いたしました。（ビジネスコンテストは優勝いたしました) 注意を引きつける効果を最大化するための光の表現を多く利用しています(フラッシュ表現や、ロゴ出現時の雷など。)

動画制作

Bynd ”6seconds Challenge

インターフェースのような素材を、立体的に合成する映像表現です。空気に映し出されているというのをより表現するために、ノイズを入れたり光沢を与えたり、様々な細かい技法を用いています。

動画制作

Kanye Remix

Remix音楽は、Logic ProXで制作している自作です。 歌詞のニュアンスに合わせ強弱やスピード・配置を細かく調整し、演出面では、和音の深さを表現するため、深海をイメージし表現しています。また、水中にある泡を表現するため、一からこだわり生成しています。「開放感」を表現するため曲が一番盛り上がる部分では、全ての靄が外れ、宇宙が背後に現れます。解放、つまり大きな空間を表す「Space(宇宙)」を用いその壮大さと開放感を表現しています。

音楽制作

Army Remix

Transitionと呼ばれる、画面を切り替える表現を多く利用したMotion動画です。この動画は、動画編集を始めた初期の作品で、内容にまとまりがありません。ですが、様々な表現を試しており、次から次へといろんな表現が含まれ変わっていきます。 RemixはLogic ProXで自作したものです。

音楽制作

ANAロジカル施工研修チラシ

研修を人事部に提案するチラシの作成ご依頼でした。短いプレゼンができる資料・当該研修のポジティブな印象の醸成をできることが目的でした。このため、表面には研修で扱う内容の簡単な問題を目を引く図と共に配置し、裏面には研修の訴求点・コース紹介を目の導線を考慮することで十分な情報を盛り込みました。結果、レイアウト/色により文字数を削減でき、情報の質/量を落とさずに完成させました。

チラシ制作
